Учебный репозиторий курса Otus C++ Developer Professional.<br>
Проектная работа

***Мессенджер***

## Цель:

закрепить полученные знания на С++;<br>
оформить работу для портфолио;<br>
применить инструментарий С++ на практике.

## Описание:

Предполагаемые уровни развития:
1. Архитектура - соединение точка-точка.<br>
Передача информации — по протоколу TCP, текст предается в обе стороны при обеспечении следующих функций:
- гарантированная доставка сообщений;
- уведомление "пользователь печатает";
- сохранение локальной истории.
2. Клиент-серверная архитектура.<br>
Взаимодействуют два экземпляра одной и той же программы — один в режиме «сервер», второй в режиме «клиент» с обеспечением следующих функций:
- регистрация;
- хранение истории пользователя на сервере.

## Требования к реализации

Работа должна быть выполнена на С++.

## Критерии завершения проектной работы

1. Созданная утилита обеспечивает передачу текста  по протоколу TCP в соответствии с описанием.

2. В реализации обеспечена клиент-серверная архитектура.
